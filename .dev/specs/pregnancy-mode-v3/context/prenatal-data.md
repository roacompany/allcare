# 한국 산전검진/바우처/산모수첩 데이터 — ③검진 탭 (의료감수 전 초안)

작성 2026-06-11 · 웹 리서치 큐레이션. **⚠️ 산부인과 전문의 감수 전 초안 — 앱 노출 전 검수 필수.** 금액·시기·기준치는 출처/연도 기준이며 정책·기관별 변동. 각 섹션 끝 '감수 주의' 반드시 확인.

## 한국 표준 산전검진 일정 (의료감수 전 초안)

> 출처: 보건복지부 임신육아종합포털 아이사랑, 대한산부인과학회(KSOG), 서울아산병원, 강남차병원, 대한진단검사의학회. 2024~2026 게재 기준. 병원·산모 위험도에 따라 시기·항목이 달라질 수 있음.

### 주차별 표준 검사 표

| 주차범위 | 검사명 | 내용/목적 | 비고 |
|---|---|---|---|
| 임신 확인~초기(약 5~10주) | 초기 기본검사 (혈액·소변) | 일반혈액검사(CBC), 혈액형(ABO/Rh), 빈혈, B형간염, 매독(VDRL), HIV, 풍진항체, 소변검사(단백뇨·당뇨·요로감염) | 첫 내원 시 시행. 풍진 항체 없으면 출산 후 접종 |
| 11주~13주 6일 | NT(태아 목덜미 투명대) + 1차 기형아(초기 통합) | 초음파로 목덜미 투명대 두께 측정 + 모체혈청 표지자(PAPP-A 등) 결합 → 다운증후군·에드워드증후군 등 염색체 이상 위험도 | NT 측정 권장 11주 0일~13주 6일. 초기·중기 결합 시 IST(통합선별검사) |
| 10~22주 (선택) | NIPT(비침습적 산전검사) | 산모 혈액 내 태아 DNA 분석으로 다운증후군 등 염색체 이상 선별 | 선택 검사(비급여). 고위험·고령 산모 권장 |
| 11~13주 (적응 시) | 융모막 융모생검(CVS) | 고령·기형아검사 이상 시 염색체 확진(침습적) | 적응증 있을 때만 |
| 15주~20주 | 쿼드검사 / 2차 기형아(통합검사) | 모체혈액 4종 호르몬(AFP·hCG·uE3·인히빈A) → 다운·에드워드·신경관결손 위험도 | 초기검사와 합치면 통합검사(정확도↑) |
| 15~20주 (적응 시) | 양수검사(양수천자) | 초음파 유도하 양수 채취로 염색체 확진(침습적) | 기형아검사 고위험·고령 등 적응증 시 |
| 20주 전후(18~24주) | 정밀초음파 (정밀 태아 구조 초음파) | 태아 주요 장기·구조적 기형 정밀 관찰(심장·뇌·척추·신장 등) | 권장 18~22주(자료에 따라 20~24주). 태아심장정밀검사 별도 가능 |
| 24주~28주 | 임신성 당뇨 선별검사 (50g 경구당부하) | 금식 없이 포도당 50g 섭취 1시간 후 채혈. 140mg/dL 이상이면 확진검사로 진행 | 모든 임산부 대상 선별 |
| 24~28주 (선별 양성 시) | 임신성 당뇨 확진검사 (100g 경구당부하) | 8시간 이상 금식 후 공복·1·2·3시간 혈당 측정. 기준치(공복95·1h180·2h155·3h140 등) 2개 이상 초과 시 임신성 당뇨 진단 | 100g 진단 기준치는 학회·기관별 약간 상이(예: 105/190/165/145 또는 95/180/155/140). 2013 대한당뇨병학회는 75g 단일검사도 허용 |
| 27주~36주 | 백일해 예방접종(Tdap) | 태아·신생아 백일해 보호를 위해 임신 중 접종 권장 | 매 임신마다 접종 |
| 32주 전후~ | NST(태아안녕검사, 비수축검사) | 태아 심박동·자궁수축 측정으로 태아 안녕 평가 | 막달·고위험 시 반복. 분만 임박 시 정기적으로 시행 |
| 35주~37주 | GBS(B군 연쇄구균) 선별검사 | 질·직장 면봉 채취 배양 → 보균 시 분만 중 항생제(IAP, 페니실린/앰피실린)로 신생아 패혈증 예방 | 모든 임산부 권장. 양성·미상 조기분만·조기양막파수 18시간↑·분만 중 발열 시 항생제 투여 |
| 35주 이후(막달) | 분만 전 검사 | 혈액검사·간기능·혈액응고검사·심전도(ECG)·흉부 X선 등 분만/마취 대비 | 막달 종합검사 |

### 정기 진찰 간격 (통념·아이사랑/KSOG 기준)

| 시기 | 진찰 주기 |
|---|---|
| 임신 ~28주 | 4주마다(월 1회) |
| 임신 28~36주 | 2주마다 |
| 임신 36(37)주~출산 | 매주 |

### 국가지원 표준 진찰 횟수
- 표준 산전 진찰 횟수는 임신 1회당 **최소 약 10~14회 수준**(상기 간격 적용 시): 28주까지 약 6~7회 + 28~36주 약 4회 + 막달 매주 약 3~4회.
- 대한산부인과학회 표기상 28주까지 4회 + 36주까지 2회 추가의 "최소 진찰" 기준도 병기됨(병원·산모 상태에 따라 가감).
- 국민행복카드 임신·출산 진료비 바우처(별도 행정 지원)는 진찰 "횟수"가 아닌 진료비 한도(단태아 100만원 등)로 지원되므로 검진 횟수 규정과는 구분 필요.

## 임신·출산 진료비 지원 (국민행복카드 임신·출산 진료비 바우처)

> 의료감수 전 초안. 모든 금액·조건은 아래 출처 기준이며, 정부 정책에 따라 변동될 수 있음. 표기 기준 연도: 2024년 확대 시행분(2025~2026 적용).

### 지원금액 (임신 1회당)
| 구분 | 지원금액 | 비고 |
|---|---|---|
| 단태아(태아 1명) | **100만 원** | |
| 다태아(태아 2명 이상) | **140만 원** | 태아 수와 무관하게 동일(쌍둥이·세쌍둥이 모두 140만 원) |
| 분만취약지 추가 | **+20만 원** | 신청 당시 분만취약지에 연속 30일 이상 거주한 임산부 |

- 예: 분만취약지 거주 단태아 → 120만 원, 다태아 → 160만 원
- (참고) 2022년 1월 이전 단태아 60만 원 / 다태아 100만 원 → 2022년 인상 → 현재 100만 원 / 140만 원

### 사용처
- **요양기관(병·의원, 산부인과 등)** 에서 진료비 결제
- **약국**: 처방받은 약제·치료재료 구입비
- **한방 요양기관** 포함(요양기관 범위 내)
- 임산부 본인 및 영유아 진료에 사용. 미용·성형, 의약외품, 건강보조식품 등은 사용 불가

### 사용 가능 항목
- 임산부의 임신·출산 관련 **진료비**(급여·비급여 본인부담금)
- 처방된 **약제·치료재료** 구입비
- **2세 미만 영유아**(출생일부터 만 24개월 미만)의 진료 및 처방 약제·치료재료 구입비 (2024년 확대: 종전 1세 미만 → 2세 미만)

### 사용 기한
- **분만예정일(출산 전 신청) 또는 출산일(출산 후 신청)부터 2년** (2024년 확대: 종전 1년 → 2년)
- 유산·사산 시 유산진단일/사산일부터 2년
- 사용기간 내 미사용 잔액은 **자동 소멸**(현금 환급 불가)

### 신청 방법
1. **요양기관(산부인과)** 에서 임신·출산 사실 확인(서면 확인서 발급 또는 온라인 입력)
2. 다음 중 한 곳에 신청:
   - **정부24**: 원스톱서비스 > 맘편한임신 온라인 통합처리 신청
   - **국민건강보험공단**: 홈페이지/The건강보험 앱 > 민원여기요 > 임신·출산 진료비 온라인 신청, 또는 공단 지사 방문
   - **카드사(금융기관)**: 국민행복카드 발급 은행/카드사 영업점 방문(BC·삼성·롯데·KB국민 등 위탁사)
3. 신청 후 국민행복카드 발급(기존 보유 시 해당 카드에 바우처 충전)

### 인접 지원 제도 (간단 참고)
- **첫만남이용권**: 출생아 1인당 첫째 **200만 원**, 둘째 이상 **300만 원**(2024년 출생아부터). 국민행복카드 바우처로 지급, 출생일(또는 출생신고일)부터 **2년** 이내 사용. 임신·출산 진료비 바우처와 별개 제도.
- **임산부 친환경농산물 지원**: 친환경 농산물 구입비의 일부(최대 약 80%) 지원. 지자체별 신청 기간·대상 상이(거주지 임산부/출산 산모). 농산물에 한정, 진료비 바우처와 별개.
- **에너지바우처·국민행복카드 기타 바우처**(아이행복카드 보육료 등)도 동일 국민행복카드로 통합 사용 가능하나 별도 신청 필요.

# 산모수첩 디지털 미러 구조 (BabyCare 임신모드 ③검진 탭)

> 전제: 모든 항목은 의료감수 전 초안. 출처는 표준 모자보건수첩(보건복지부/정부24), 대한산부인과학회(KSOG), 서울아산병원, 강남차병원, 질병관리청, 국민건강보험공단 공개 자료 기준(2023~2026 확인분).
> 입력 주체 구분 표기: **[앱]** = 임산부가 직접 입력 가능 / **[병원]** = 의료진이 측정·판독하는 값(임산부가 검진 후 받아 적는/스캔하는 항목) / **[자동]** = 앱이 계산·파생(예: 임신주수, 추세 그래프).

## 섹션 2. 산과력 / 병력 (Obstetric & Medical History)

**가족력** — [앱] 입력 (체크리스트)

**생활/노출력** — [앱] 입력

## 섹션 3. 매 검진 기록 (Antenatal Visit Log) — 핵심 반복 엔티티

검진 주기 기준(KSOG/일반 산과): 임신 28주까지 4주마다 → 36주까지 2주마다 → 이후 매주.

| 필드 | 타입 | 주체 | 비고 |

## 섹션 5. 혈액형·감염·선별 검사 결과 (Lab & Screening Results)

대부분 임신 중 1회성(결과 누적형). 출처: 표준 모자보건수첩 검사항목 + 서울아산병원 + 강남차병원 + 대한진단검사의학회.

**기본/감염 검사** (초기 혈액검사, 임신 확인 시) — [병원] 결과 입력
- 혈액형/Rh · 일반혈액검사(빈혈/Hb) · 혈당 · B형간염(HBsAg) · 매독(VDRL/RPR) · HIV · 풍진항체(Rubella IgG) · 갑상선기능 · 소변검사

**기형아/염색체 선별** — [병원]
- 목덜미투명대(NT) 11~13주6일
- 통합/쿼드검사(혈청표지자) 15~20주 (검출률 약 50~70% 참고)
- NIPT(비침습 산전검사) / 양수검사(필요 시 15~20주)

**중·후기 선별** — [병원]
- 임신성당뇨 선별(50g GCT) 24~28주, 양성 시 100g OGTT 정밀
- 후기 빈혈 재검 / 혈액형 재검 24주 이후
- GBS(B군 연쇄상구균) 35~37주 (질·직장 도말, 약 25% 보균)
- NST(비수축검사) 임신 후기(보통 32~40주, 병원별 상이)

각 검사 레코드: `검사명 / 검사일 / 주수 / 결과값 또는 정상·이상 / 참고치 / 재검 여부 / 첨부` — 결과값은 **[병원]**, 검사 받았다는 사실·날짜·사진은 **[앱]** 입력 가능.

## 설계 권고 요약

1. **반복 엔티티 3종**(검진 로그 §3 / 초음파 §4 / 검사결과 §5)은 시계열 컬렉션으로, **고정 1회성**(기본정보 §1 / 산과력 §2 / 분만 §7)과 분리.
2. **[병원] 값은 "수치 직접 입력" + "사진 스캔 보관"을 모두 허용** — 임산부가 OCR 없이도 사진으로 보존 가능하게.
3. **알림 트리거 후보**: 검진 주기(4/2/1주), Tdap 27~36주, 독감 시즌, GBS 35~37주, Rh(−) 항-D, 혈압 140/90·단백뇨 병발 경고.
4. **추세 시각화 우선순위**: 체중(권장범위 밴드) > 혈압 > 자궁저높이 vs 주수 > 태아심음.
5. **의료 면책**: 모든 참고치(FHR 110~160, 혈압 140/90, 자궁고 ±2cm 등)는 일반 참고용 문구 병기 필요.
